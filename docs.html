API Freshstart

L’API Freshstart fonctionne uniquement en JSON.

Elle est constituée de 2 API de haut niveau : Courses et Coureurs.

Sécurité ¶

L’accès à l’API Freshstart est contrôlé et limité. Pour accéder à l’API, vous devez posséder une clé d’API. Cette dernière prend la forme d’un couple de UUID Application ID et Application Secret. Vous pouvez obtenir cette clé auprès de votre interlocuteur habituel de Freshstart.
